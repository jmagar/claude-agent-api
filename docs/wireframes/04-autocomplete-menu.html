Key Features Shown:

  • @ Trigger: Universal autocomplete for agents, MCP servers, tools, resources, files, skills, presets
  • / Trigger: Commands and skills only
  • Categorization: Items grouped and tagged by type (Agent, MCP Server, MCP Tool, File, Skill)
  • Recently Used: Frequently accessed items appear at top for quick access
  • Search Filtering: Type to filter results in real-time (e.g., "@pos" shows postgres items)
  • Keyboard Navigation: Full keyboard support with shortcuts shown in footer
  • Attach Button: Opens menu for uploading images/files or selecting from project
