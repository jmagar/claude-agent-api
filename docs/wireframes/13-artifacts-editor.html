Artifacts Editor Features:

  • Right Slide-in Panel: Covers 55% of screen from right side
  • Multiple Tabs: Switch between multiple generated artifacts
  • Three View Modes: Editor only, Preview only, Split view (both)
  • Syntax Highlighting: Color-coded by language (TS, JS, Python, SQL, etc.)
  • Live Preview: Real-time rendering for React components, HTML, diagrams
  • Code Actions: Download, Copy to clipboard, Format, Lint
  • Save to Project: In Code mode, writes artifact directly to filesystem
  • Language Detection: Auto-detects language from file extension
  • Status Bar: Shows cursor position, encoding, language, errors/warnings
  • PlateJS Integration: Full rich-text editing capabilities

Triggered When:

  • Agent generates code, diagrams, or documents
  • User clicks "View" on inline artifact preview in chat
  • User asks to "edit the dashboard component"
  • Multiple artifacts accumulated during session

Supported Artifact Types:

  • Code files (React, TypeScript, Python, SQL, etc.)
  • Diagrams (Mermaid, PlantUML)
  • Documents (Markdown with live preview)
  • Configuration files (JSON, YAML)
