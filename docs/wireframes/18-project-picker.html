Project Picker Features:

  • Modal Picker: Full-screen modal for selecting/creating projects
  • Search: Real-time filtering of projects by name or path
  • Project Cards: Rich metadata including file count, last accessed, git branch
  • Technology Badges: Visual indicators for Git, Python, Node.js, Docker, etc.
  • Inline Picker: Dropdown in Code mode topbar for quick switching
  • Quick Actions: Open in VS Code, Terminal, or Project Settings
  • Create New: Templated project creation with optional Git init
  • Empty State: Friendly onboarding for first-time users
  • Hover Actions: Edit and delete buttons appear on hover
  • Active Indicator: Checkmark shows currently selected project

UX Flow:

  • User switches to Code mode ‚Üí Project picker modal appears (if no project selected)
  • User clicks inline picker dropdown ‚Üí Modal opens with search and recent projects
  • User searches for project ‚Üí Real-time filtering with highlighted matches
  • User clicks project card ‚Üí Modal closes, project loads, filesystem tools become available
  • User clicks "New Project" ‚Üí Create project form with template options
  • Project metadata updates automatically (last accessed, file count)

Technical Notes:

  • Project metadata stored in PostgreSQL (name, path, last_accessed, file_count, git_branch)
  • File count updated via filesystem watcher or periodic scan
  • Git status checked via git CLI commands (current branch, dirty state)
  • Technology badges detected from package.json, pyproject.toml, Dockerfile, etc.
  • Templates scaffold new projects with sensible defaults
