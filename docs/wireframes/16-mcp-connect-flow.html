Inline /mcp connect Flow Features:

  • Slash Command Trigger: User types /mcp connect in chat input
  • Quick Config Form: Minimal fields for fast setup (name, type, command, args)
  • Template Support: Pre-configured templates for common servers (PostgreSQL, GitHub, etc.)
  • Connection Types: stdio, SSE, HTTP with type-specific fields
  • Test Connection: Validate configuration before saving
  • Save to Library: Optional checkbox to add to permanent server list
  • Auto-Connect: Optional checkbox to connect in future sessions
  • Inline Feedback: Success message in chat with exposed tools list
  • Error Handling: Clear error messages with suggestions

Variants:

  • /mcp connect postgres - Quick connect with template name
  • /mcp connect [URL] - Direct SSE/HTTP connection
  • /mcp add - Alias for /mcp connect
  • /mcp disconnect [name] - Remove from current session
  • /mcp list - Show all connected servers

Success Criteria:

  • Server connects successfully and exposes tools
  • Tools immediately available via @ autocomplete
  • Session remembers connection (persists to DB)
  • If "Save to library" checked, appears in settings admin panel
