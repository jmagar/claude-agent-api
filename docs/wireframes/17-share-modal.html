Share URL Generation Features:

  • Shareable Content: Agents, skills, slash commands, MCP configs, sessions
  • Credential Sanitization: Automatically removes API keys, tokens, passwords
  • View-Only Links: Recipients can view but not edit (unless "Allow forks" enabled)
  • Download Options: Copy link, download .md/.json, copy raw content
  • Privacy Controls: Optional password protection, expiration dates
  • Content Options: Include/exclude thinking blocks, tool calls, subagents
  • Preview: Shows exactly what will be shared before generating link
  • Security Warnings: Alerts when credentials detected, requires sanitization

Use Cases:

  • Share agent configurations with teammates
  • Distribute MCP server setups (with sanitized credentials)
  • Share conversation examples for documentation
  • Create public library of reusable configs
  • Export sessions for troubleshooting/support

Security Features:

  • Credential sanitization mandatory for MCP configs
  • Pattern matching for common secret formats (API keys, tokens, passwords)
  • Preview shows exactly what will be shared
  • Optional password protection for sensitive sessions
  • Expiration dates to limit link lifetime
