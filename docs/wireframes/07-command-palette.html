Command Palette Features:

  • Unified Interface: Single `Cmd+K` modal for both commands (actions) and search (finding things)
  • Real-Time Search: Filter results as you type across all categories
  • Categorized Results: Commands, MCP Servers, Agents, Sessions, Settings organized into sections
  • Status Indicators: Shows connection status for MCP servers, badges for entity types
  • Keyboard Shortcuts: Navigate with arrows, select with Enter, close with Esc
  • Contextual Info: Each item shows relevant metadata (tool count, timestamps, descriptions)
  • Quick Actions: Direct access to frequently used commands without leaving current view

Search Scope:

  • Commands and slash commands
  • MCP servers and their tools/resources
  • Agents and skills
  • Conversation history (sessions)
  • Settings and configuration pages
  • Files (in Code mode)
