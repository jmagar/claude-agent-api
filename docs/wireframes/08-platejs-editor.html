PlateJS Editor Features:

  • Right-Side Slide-in: Editor appears from right, covering 60% of screen
  • YAML Frontmatter: Structured fields for name, description, tools, skills
  • Markdown Body: Full-featured editor for system prompt/instructions
  • Tool Preset Selector: Choose from saved tool configurations
  • Preview Pane: Shows how agent appears in autocomplete menu
  • Tabs: Edit, Preview (rendered markdown), Version History (with diffs)
  • Share Button: Generate view-only URL with credential sanitization
  • Actions: Save, Cancel, Delete, Share with confirmation dialogs

Use Cases:

  • Creating/editing custom agents
  • Configuring skills with markdown documentation
  • Writing slash commands with YAML + markdown
  • Editing MCP server configs (JSON with schema validation)
  • Sharing configurations with team (auto-sanitized credentials)
