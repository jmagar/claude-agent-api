Settings Panel - All 8 Sections:

  • 1. Account: API key management, usage statistics dashboard
  • 2. Appearance: Light/Dark/System theme, font size, message density
  • 3. Visualization: Threading display mode, thinking blocks, tool card auto-expand
  • 4. Defaults: Default permissions mode, tool preset, workspace directory
  • 5. MCP Servers: Links to admin panel for server configuration
  • 6. Agents & Skills: Browse, create, edit agents/skills/commands
  • 7. Keyboard Shortcuts: View and customize all hotkeys
  • 8. Advanced: Debug mode, logs viewer, reset settings, telemetry

Key Features:

  • Two-panel layout: sidebar navigation + content area
  • Accessible via Cmd+K ‚Üí "Settings" or gear icon
  • Live preview of setting changes where applicable
  • Organized into logical sections with clear descriptions
  • Toggle switches for binary options, dropdowns for multi-choice
