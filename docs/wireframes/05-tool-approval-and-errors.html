Permissions Chip Modes
Plan Mode
No auto-execution. Agent shows implementation plan and waits for approval before taking any action.
Edit Automatically
Auto-approve all tools including file edits and writes. Best for trusted workflows.
Ask before Edits
Auto-approve reads. Require approval for writes and modifications. Balanced safety.
YOLO (Bypass All)
Auto-approve everything with no confirmations. Use with caution!

Key Features Shown:

  • Inline Blocking Approval: Tool card appears in message flow with Approve/Deny buttons
  • Detailed Parameters: Shows exactly what the tool will do (file path, changes, etc.)
  • "Always Allow" Option: Checkbox to auto-approve specific tool for current session
  • Error Cards: Red-bordered cards with error message, collapsible stack trace, and retry button
  • Contextual Error Messages: Agent responds to errors with helpful explanations and next steps
  • Four Permissions Modes: Plan Mode, Edit Automatically, Ask before Edits, YOLO
